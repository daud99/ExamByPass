# ExamByPass

For Bahaa and Khaleed.

## For development

### Running the server
```sh
cd server
```
```sh
npm install
```
```sh
npm start
```

### Running the vue app for development
```sh
cd client
```
```sh
npm install
```
```sh
npm start
```

> By above way we will be able to test both node and vue running on single port 3000

### Running the vue app for production
```sh
cd client
```
```sh
npm install
```
```sh
npm run build
```
